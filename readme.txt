# 伺かゴースト「You save me, I love you.」

扉の向こうから響く、やさしいやさしい弟の声。


## 成分表示

* ヤンデレな弟（「ヤンデレ」は作者独自の解釈を含みます）
* ユーザに少し設定があります
  * 「喋らない」「部屋から出ない」
  * 「過去に触れられたくない記憶がある」ような描写はありますが、詳細は語られません
* 恋愛要素・性的描写：なし
* 暴力表現：直接的描写なし。間接的示唆のみ。
* 状態・関係性の変化：なし


## これはなに

ゴーストという種類のデータセットです。
動かすには、「SSP」というソフトを使うのがおすすめです。

* SSPの配布元
  * https://ssp.shillest.net/

* 基本的な使い方
  * https://keshiki.nobody.jp/


## 使用方法詳細
* 喋り頻度変更など
  * ドアノブをダブルクリック

* 何か話してもらう
  * ドアをダブルクリック
